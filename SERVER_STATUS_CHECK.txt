CHECK SERVER STATUS - RUN IN SSH
================================

Since your server should be running, let's verify its status:

# 1. Check if Node.js process is actually running
ps aux | grep node | grep -v grep

# 2. Check if the server is responding locally
curl -v http://localhost:8080/api/dashboard
curl -v http://127.0.0.1:8080/api/dashboard

# 3. Check what's listening on port 8080
netstat -tlnp | grep :8080
lsof -i :8080

# 4. Check server logs
ls -la *.log
tail -f simple.log 2>/dev/null || tail -f server.log 2>/dev/null || echo "No log files found"

# 5. If server died, restart it
nohup node simple.js > server.log 2>&1 &

# 6. Test the specific endpoints
curl http://localhost:8080/
curl http://localhost:8080/api/dashboard

If localhost:8080 works but opt.vivaindia.com doesn't, it's definitely a nginx proxy configuration issue.
