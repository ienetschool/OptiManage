üìù Plesk Configuration for Direct Access (No Port Numbers)

  1. Open Plesk control panel for opt.vivaindia.com
  2. Go to Apache & nginx Settings
  3. Enable nginx for this domain
  4. In nginx directives, add this configuration:
location / {
    proxy_pass http://localhost:8080;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
    proxy_buffering off;
}
  1. Click Apply to save settings
  2. Test access at http://opt.vivaindia.com (no port needed)

Result: Clean access at http://opt.vivaindia.com with no port numbers or redirects
